# Advent of Code 2025

Files for the 2024 [Advent of Code](https://adventofcode.com/) programming challenge. Specific input files are not included, but can be found on the website. 

# Requirements

* Python 3.14+
* `uv` package manager

# Running Specific Solutions

All code is run through `main.py`. Import the correct module and run `main` with the appropriate input placed in `/input.txt`.
